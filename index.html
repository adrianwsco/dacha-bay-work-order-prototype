About This Screen

Field operations dashboard. Each card shows the turbine number, job type, and current status for active work orders at Dacha Bay Wind Farm.

WO-2847 is the live demo. Click it to walk through the full workflow.

Patterns: multi-screen workflow, status tracking, human-in-the-loop design.

About This Screen

The technician's field form. Each section maps to a real AI implementation pattern:

Progress Tracker

Gates submission until all required fields are complete. Nothing can be skipped.

Photo Upload + AI Vision

Upload a blade photo and Claude reads the visible damage and pre-fills the erosion category. To test it, download a real photo from Wikimedia Commons: Leading edge damage (876x657 jpg).

LEE Assessment

Structured data capture. Damage category and erosion length per blade. This data feeds the field notes generator.

✦ Generate Draft Notes

Calls the Claude API live, reads everything the technician filled in, and writes a professional field report. The output is editable before submission.

Cat 5 Auto-Escalation

If any blade is flagged Cat 5, the system drafts an escalation email in the connected Gmail account, ready to review and send.

Patterns: multimodal AI input (vision), AI text generation, structured data capture, condition-triggered automation, editable AI output.

About This Screen

Submission complete. The structured form data is packaged as a JSON payload and sent to a Google Apps Script endpoint, which writes it to Google Drive. Blade photos from the inspection are saved to the same folder, organized by work order number.

Open Google Drive in a second tab and refresh after submitting. The folder, the JSON file, and any uploaded photos appear in real time.

Patterns: structured output, API integration, data persistence, end-to-end workflow.

About This Screen

The site manager's view. Read-only overview of all active work orders across the wind farm: turbine, job type, technician, and current status.

✦ Run AI Triage

Sends all open work orders to the Claude API in a single call. Claude returns a ranked priority list (P1 through P4) with a rationale for each. The output is advisory only. No work orders are modified automatically.

✦ Generate Shift Report

Compiles all work orders into a structured end-of-shift summary written in plain language. The report is editable before it is sent or saved.

Patterns: AI decision support, batch structured output, stateless triage, AI document generation, human-in-the-loop.
